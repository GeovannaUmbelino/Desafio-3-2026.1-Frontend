# A.N.G.E.L.I.C.A — Dashboard de Análise Territorial

Painel desenvolvido para a **A.N.G.E.L.I.C.A** (Agência Nacional de Gerenciamento de Estados e Logística de Informações Continentais para Análise). Este projeto centraliza a consulta de dados geográficos brasileiros, substituindo planilhas fragmentadas por uma interface intuitiva e dinâmica.

Este projeto foi desenvolvido durante minha **Capacitação Técnica como Trainee na Engnet**, consolidando conhecimentos em consumo de APIs, arquitetura de front-end e design de interface.

## 🚀 Funcionalidades

- **Navegação Inteligente:** Filtro de estados (UFs) por região e visualização em *accordion* no corpo da página.
- **KPIs Dinâmicos:** Cards de resumo (Cidades, ID da UF, Região, IDH e Concentração Logística) atualizados em tempo real.
- **Busca Eficiente:** Filtro de municípios na tabela com busca local (sem novas requisições à API).
- **Integração Logística:** Acesso direto via Google Maps ao clicar no nome de qualquer município.
- **Métricas Territoriais:** Visualização gráfica de UFs por região e indicadores de densidade demográfica.

## 🛠️ Tecnologias Utilizadas

- **Front-end:** HTML5, CSS3, JavaScript.
- **API:** [API de Localidades do IBGE](https://servicodados.ibge.gov.br/api/docs/localidades?versao=1).
- **Bibliotecas:** [Chart.js](https://www.chartjs.org/) para visualização de dados.
- **Design:** Prototipagem em média fidelidade no Figma, seguindo rigoroso guia de estilos.
